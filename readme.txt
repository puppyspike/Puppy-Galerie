Tags: puppy, galerie

Galerie Menü für die Puppy Galerien

== Description ==
Das ist das Plugin, um in WordPress ein Galerie Menü für die Puppy Galerien zu erhalten.

Hier kann man dann die gewünschten Galerien anlegen und verwalten.
